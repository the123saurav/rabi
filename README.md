# rabi
A LSM based key value database in Java.

# Project Status
Implementation is going on.
At the moment:
- PUT is working.
- Boot is working.
- Flushing to disk is working.

# Building project
This project uses maven for builds. 
Requirements:
- maven CLI(tested with 3.6.2)
- Java 8
